# grc-engineering-cgep-labs - Working Repository

Hands-on lab work for the **Certified GRC Engineer – Practitioner (CGE-P)**.
Each lab builds a reusable, compliant-by-construction GRC engineering
primitive — infrastructure-as-code, policy-as-code, evidence pipelines —
that later assembles into the capstone.

## Layout
- `terraform/primitives/` — reusable compliant IaC building blocks (one dir per lab)
- `policies/` — Rego policy suites (Domain 3 labs)
- `evidence/` — machine-readable compliance evidence output

## Reference
Answer keys / runnable reference workspaces live in `GRCEngClub/cgep-labs`
(cloned separately, read-only).
